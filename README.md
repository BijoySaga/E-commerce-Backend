# E-commerce Backend (Work in Progress)

A production-style backend system built using Java and Spring Boot.

## Tech Stack
- Java
- Spring Boot
- MySQL
- JWT Authentication
- Docker (planned)

## Planned Features
- User authentication with JWT
- Role-based authorization (Admin/User)
- Product catalog APIs
- Shopping cart management
- Order processing
- Global exception handling

## Architecture (Planned)

Client → REST API → Service Layer → Repository Layer → MySQL Database

## Status
Currently under development as part of backend engineering portfolio.
